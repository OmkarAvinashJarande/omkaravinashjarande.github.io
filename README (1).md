# Omkar Jarande – Personal Portfolio

Personal portfolio website hosted via GitHub Pages.

**Live:** [https://omkar-jarande.github.io](https://omkar-jarande.github.io)

## Structure

```
├── index.html      # Main HTML
├── style.css       # Styles
├── main.js         # Animations & interactivity
├── resume.pdf      # Your CV (add this yourself)
└── README.md
```

## How to Update

1. Edit `index.html` to update content
2. Push changes to the `main` branch
3. GitHub Pages will redeploy automatically

## Setup

1. Go to your GitHub repo → **Settings → Pages**
2. Under **Source**, select `main` branch and `/ (root)` folder
3. Save — your site will be live at `https://<your-username>.github.io/<repo-name>`

If the repo is named exactly `<your-username>.github.io`, it'll be live at `https://<your-username>.github.io`.
